1. git clone "path - to- repositoriy" клонирует проект с гитхаба в текущую папку
2. git status - проверяет наличие новых файлов.
3. git add path-to-file - добовляет новый файл под контроль git.
4. git commit -am "comment" -добавление коментариев к своим действиям
5. git push - отправка закомиченных файлов на сервер github
6. git pull - забрать свежие обнавления с сервера github
